Input Panel

Number of Trials (n) Fixed number of independent trials/experiments. Each trial has the same success probability p.
n ∈ {1, 2, 3, ...}, mean = np
n=10: 10 coin flips, n=20: 20 survey responses, n=100: 100 tests
Success Probability (p) Probability of success on each individual trial. Constant across all n trials.
0 ≤ p ≤ 1, variance = np(1-p)
p=0.5: fair coin, p=0.1: rare success, p=0.8: likely success

Results Panel - Visualizations & Statistics

PMF View
Probability Mass Function Shows discrete probabilities for exact numbers of successes. Bars show P(X = k).
Height = probability of exactly k successes in n trials
CDF View
Cumulative Distribution Function Shows cumulative probability P(X ≤ k). Staircase pattern from 0 to 1.
Each step up represents adding one more possible success
Combined View
PMF + CDF Together Compare discrete probabilities (PMF) and cumulative probabilities (CDF) side by side.
See how individual probabilities accumulate to total probability
